Project Overview

This project demonstrates ETL pipeline and analytics workflow using PySpark on Databricks. The goal of the project is to ETL pipeline to analyze sales transactions and generate meaningful business insights such as customer spending, product performance, time-based trends, and sales distribution across locations and order sources.

The project uses structured schemas, Spark SQL functions, and aggregations to process and analyze data efficiently.
